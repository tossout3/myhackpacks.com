My gearbadge diploma level 1

A curated collection of my favorite hack demos and walkthrough videos.

HackPack 1 / IR Turret
In this HackPack I added IR patterns for each type of controller (e.g. NEC, NEC2) that send IR signals.
HackPack 2 / Domino Robot
In this HackPack I added a diagonal proximity (distance) sensor to detect if it's about to fall, so it can stop in time when the distance from it to the ground is too big.
HackPack 3 / Label Maker
In this hackpack I attached a USB host to connect to a keyboard I replaced the function keys at the top with symbols like Mark Robers logo. To make the code not have to store all the symbols and letters I added a WI-FI module to access a Github website that I made so the code tells the symbol or letter then the Github website gives the movements using JS for the servo and stepper motors.
HackPack 4 / Sand Garden
In this Hackpack I didn't do any hardware hacks instead I changed the code so both the stepper motors are async because if they are in sync it becomes very difficult to draw a straight line. It will only be sync if the user is using the code's sand garden image.
